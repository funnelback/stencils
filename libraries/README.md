## Introduction
All the stencils files.

## Stencils

Stencils can be categorised as a _'Search Configuration'_ or _'Standalone Component'_.

A _'Search Configuration'_ stencil encompasses utilities, example views and an example search configuration for a specific holistic set of requirements for an entire search application.

A _'Standalone Component'_ stencil encompasses utilities, example views that can fit within _'Search Configuration'_ stencils.
